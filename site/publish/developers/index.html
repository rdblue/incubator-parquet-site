Developers

Report or track a bug

New bugs can be reported on our Jira issue tracker. In order to create a new issue, you’ll need to signup for an account.

Contribute a core patch

Follow our contribution guidelines when submitting a patch.

IRC Channel

Many of the Aurora developers and users chat in the #aurora channel on irc.freenode.net.

If you are new to IRC, you can use a web-based client.
